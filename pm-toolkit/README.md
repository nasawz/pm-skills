# PM工具包

PM实用技能：简历审核、NDA起草、隐私政策生成以及语法/流畅度检查。产品经理除核心产品工作之外必备的工具。

## 技能（4项）

- **draft-nda** — 起草双方之间的详细保密协议（NDA）。
- **grammar-check** — 识别文本中的语法、逻辑和流畅度错误，并在不重写整个文本的情况下建议有针对性的修正。
- **privacy-policy** — 为产品起草详细的隐私政策，涵盖数据类型、司法管辖区、合规考虑事项以及需要法律审查的条款。
- **review-resume** — 根据10项最佳实践对PM简历进行全面的审核和定制，包括XYZ+S公式、关键词优化、职位特定定制和结构优化。

## 命令（5条）

- `/pm-toolkit:draft-nda` — 起草双方之间的保密协议，包含适用于特定司法管辖区的条款。
- `/pm-toolkit:privacy-policy` — 起草隐私政策，涵盖数据收集、使用、存储和合规要求。
- `/pm-toolkit:proofread` — 检查任何文本的语法、逻辑和流畅度 —— 提供有针对性的修正而不重写全文。
- `/pm-toolkit:review-resume` — 根据十项最佳实践全面审核PM简历 —— 结构、影响力指标、关键词和可操作的反馈。
- `/pm-toolkit:tailor-resume` — 根据特定职位描述定制PM简历 —— 关键词对齐、经验重构和战略优化。
