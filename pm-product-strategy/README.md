# pm-product-strategy

产品经理产品战略技能：愿景、战略画布、创业画布、价值主张、精益画布、商业模式画布、SWOT、PESTLE、安索夫矩阵、波特五力、定价和变现。

## 技能 (12)

- **ansoff-matrix** — 生成安索夫矩阵分析，映射市场渗透、市场开发、产品开发和多元化的增长战略。
- **business-model** — 生成包含所有9个构建块的商业模式画布。
- **lean-canvas** — 生成精益画布商业模式，包含问题、解决方案、指标、成本结构、价值主张、不公平优势、渠道、细分市场和收入等部分。
- **monetization-strategy** — 头脑风暴3-5种变现策略，包括受众匹配度、风险和验证实验。
- **pestle-analysis** — 进行PESTLE分析，涵盖政治、经济、社会、技术、法律和环境因素。
- **porters-five-forces** — 进行波特五力分析，评估竞争强度、供应商议价能力、买方议价能力、替代品威胁和新进入者威胁。
- **pricing-strategy** — 分析和设计定价策略，包括定价模型、竞争定价分析、支付意愿估算和价格弹性考虑。
- **product-strategy** — 使用9部分的产品战略画布生成全面的产品战略，涵盖愿景、细分市场、成本、价值主张、权衡、指标、增长、能力和防御性。
- **product-vision** — 头脑风暴一个鼓舞人心、可实现且富有情感的产品愿景，以激励团队。
- **startup-canvas** — 为新产品生成创业画布，结合产品战略（9个部分）和商业模式（成本结构+收入流）。这是商业模式画布和精益画布的替代方案，将战略与商业模式区分开来。
- **swot-analysis** — 进行详细的SWOT分析，识别优势、劣势、机会和威胁，并提供可操作的建议。
- **value-proposition** — 使用6部分的JTBD模板（谁、为什么、之前是什么、如何、之后是什么、替代方案）生成详细的价值主张。

## 命令 (5)

- `/pm-product-strategy:business-model` — 使用精益画布、商业模式画布、创业画布或价值主张框架探索商业模式。
- `/pm-product-strategy:market-scan` — 全面的宏观环境分析——在一次扫描中完成SWOT、PESTLE、波特五力和安索夫矩阵分析。
- `/pm-product-strategy:pricing` — 设计定价策略——模型、竞争分析、支付意愿估算和定价实验。
- `/pm-product-strategy:strategy` — 使用9部分战略画布创建全面的产品战略——从愿景到防御性。
- `/pm-product-strategy:value-proposition` — 使用6部分的JTBD模板设计价值主张——谁、为什么、之前是什么、如何、之后是什么、替代方案。
