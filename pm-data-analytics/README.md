# pm-data-analytics

产品经理数据分析技能：SQL查询生成和队列分析。分析用户数据、生成查询语句，并识别用户留存模式。

## 技能 (3)

- **ab-test-analysis** — 分析A/B测试结果，包括统计显著性检验、样本量验证、置信区间计算，以及可执行的建议。
- **cohort-analysis** — 对用户参与数据进行队列分析。
- **sql-queries** — 根据自然语言描述生成SQL查询语句。

## 命令 (3)

- `/pm-data-analytics:analyze-cohorts` — 对用户数据进行队列分析——留存曲线、功能采用率和参与度趋势。
- `/pm-data-analytics:analyze-test` — 分析A/B测试结果——统计显著性检验、样本量验证，以及发布/扩展/停止的建议。
- `/pm-data-analytics:write-query` — 根据自然语言生成SQL查询——支持BigQuery、PostgreSQL、MySQL等多种数据库。
