# 产品上市策略（Go-to-Market）

产品经理的上市技能：GTM策略、增长循环、GTM动作、滩头细分市场以及理想客户画像。

## 技能（6项）

- **beachhead-segment** — 识别产品发布的首个滩头市场细分。
- **competitive-battlecard** — 创建销售就绪的竞争对手对战卡，将您的产品与特定竞争对手进行比较。
- **growth-loops** — 识别可持续增长的增长循环（飞轮）。
- **gtm-motions** — 识别最佳的GTM动作和工具。
- **gtm-strategy** — 为产品发布创建上市策略，涵盖营销渠道、信息传递、成功指标和发布计划。
- **ideal-customer-profile** — 根据研究数据识别理想客户画像（ICP），包括人口统计、行为特征、待办任务和需求。

## 命令（3个）

- `/pm-go-to-market:battlecard` — 创建销售就绪的竞争对手对战卡 — 市场定位、功能对比、异议处理和制胜策略。
- `/pm-go-to-market:growth-strategy` — 设计可持续增长机制 — 增长循环和GTM动作，适用于产品驱动和销售驱动策略。
- `/pm-go-to-market:plan-launch` — 创建完整的上市策略 — 滩头细分市场、理想客户画像、信息传递、渠道和发布计划。
