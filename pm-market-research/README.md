# pm-market-research

面向产品经理的市场研究技能：用户画像、市场细分、情感分析和竞争分析。

## 技能（7个）

- **competitor-analysis** — 分析竞争对手，包括优势、劣势和差异化机会。
- **customer-journey-map** — 创建端到端的客户旅程地图，包含阶段、接触点、情绪、痛点和机会。
- **market-segments** — 识别3-5个潜在的客户细分市场，包含人口统计学特征、待办任务（JTBD）和产品契合度分析。
- **market-sizing** — 使用TAM、SAM和SOM，结合自上而下和自下而上的方法估算市场规模。
- **sentiment-analysis** — 分析用户反馈数据，识别具有情感评分、待办任务（JTBD）和产品满意度洞察的市场细分。
- **user-personas** — 从研究数据中创建精细化的用户画像。
- **user-segmentation** — 根据行为、待办任务（JTBD）和需求，从反馈数据中细分用户。

## 命令（3个）

- `/pm-market-research:analyze-feedback` — 大规模分析用户反馈 — 情感分析、主题提取和细分层面的洞察。
- `/pm-market-research:competitive-analysis` — 分析竞争格局 — 识别竞争对手、比较优劣势、发现差异化机会。
- `/pm-market-research:research-users` — 全面用户研究 — 从研究数据中构建用户画像、细分用户并绘制客户旅程地图。
